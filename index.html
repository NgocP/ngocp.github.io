United States COVID-19 Cases

Ngoc Pham - CSC196V - Interactive Design Assignment

Description
Dataset

For this assignment, I chose the dataset from the midterm: https://github.com/CSSEGISandData/COVID-19/tree/master/csse_covid_19_data/csse_covid_19_time_series. The reason why I chose this dataset is because of how the Corona Virus is affecting our everyday's life. More and more human's life is at risk, especially in the U.S where the number of infected cases ranks the highest in the world. The data starts from January 22nd, but I chose to start the date range on March 1st, 2020 because that is when the number of infected cases in the U.S starts to increase.

Visualization

I want to focus on the United States as the visualization, so using a coloring map seems like the best choice here. I let the user choose the date to view the data in the map. I also go beyond and run a simulation to see how the map color, which is the number of infected cases, change over time. This assignment and this visualization gives a sense of how serious the COVID-19 is, and I hope that people take this problem seriously and stay home.

Development Time and Progress

I spent roughly 5 days, or about 30 hours to develop this application. I think that D3 has a deep learning curve, especially when I don't have much experience with JavaScript. The most time consuming task is to figure out how to change the map color when the input date is different. This task took me about 3 days to figure it out.
